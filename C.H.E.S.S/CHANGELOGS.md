## v0.1.0-pre-alpha

- Initial release of the project.
- Heuristic model with good old tree search.
- Optimizations include alpha-beta pruning, transposition checking (using Zobrist hashing), and move ordering with MVV-LVA.
- Modes have PvP, PvC, and Perft Test.
- PvP is classic Player Vs Player with two players.
- PvC is Player Vs Computer with the heuristic engine with all the optimizations.
- Perft Test is performance analysis for checking the board rules programming.
